# volden
Volden is a python tool for extracting the Number density map of interstellar filaments from their respective column density map.

For our paper, we used the following versions:

`Python Version: 3.9.18`

`Dependencies`:
- RadFil
- Matplotlib (3.6.3)
- Numpy (1.24.4)
- Astropy (5.0.1)
- Scipy (1.8.0)
- Pandas (1.4.0)
- shapely (1.8.1)
- networkx (2.7)
- scikit-learn (1.1.0)
- scikit-image (0.19.2)
- seaborn (0.13.1)
- descartes (1.1.0)
